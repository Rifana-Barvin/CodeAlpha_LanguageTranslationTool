CodeAlpha Language Translation Tool

📖 Overview

The CodeAlpha Language Translation Tool is a Python-based application that allows users to translate text between multiple languages. It features a simple GUI with dropdown menus, copy functionality, and speech output for accessibility.

🚀 Features

Translate text between multiple languages

User-friendly GUI built with Tkinter

Dropdown menus for language selection

Copy translated text to clipboard

Speech synthesis for listening to translations

🛠️ Installation

Clone the repository:

git clone https://github.com/Rifana-Barvin/CodeAlpha_LanguageTranslationTool.git

Navigate to the project folder:

cd CodeAlpha_LanguageTranslationTool

Install dependencies:

pip install -r requirements.txt

📂 Usage

Run the application:

python main.py

Example Workflow

Enter text in the input box.

Select source and target languages from dropdown menus.

Click Translate.

Copy or listen to the translated text.
